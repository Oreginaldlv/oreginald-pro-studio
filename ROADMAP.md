# Oreginald Pro Studio Roadmap

## Phase 1 (Current MVP)
- App shell with dark UI and transport.
- Arrangement + Session workspace tabs.
- MIDI note entry and clip serialization.
- Audio device lifecycle and synth playback.
- Project save/load base.
- Browser indexing/search base.

## Phase 2 (Creative tools)
- Full MIDI editing (move/delete/stretch, snap grid, quantize).
- Drum sampler pads with WAV assignment and per-pad amp envelope.
- Track inspector with mute/solo/pan/fader/arm and color labels.
- Device rack UI for Gain/Delay/Reverb/Saturation.
- Recording lane creation + waveform clip rendering.

## Phase 3 (AI + stem + live)
- Stem separation engine integration.
- Similarity search in browser.
- MIDI generation + transformation suite.
- Scale/key assistant workflow.
- Performance mode scene packs and clip actions.

## Phase 4 (Plugin ecosystem)
- VST3 hosting and sandboxing.
- Scriptable device layer (max-for-live style).
- Marketplace-ready packaging and signing pipeline.
